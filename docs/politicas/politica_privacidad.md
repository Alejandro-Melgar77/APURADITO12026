# POLÍTICA DE PRIVACIDAD — APURADITO

**Última actualización:** Julio 2026  
**Versión:** 1.0

## 1. Datos que Recopilamos

### Datos proporcionados por el usuario:
- Nombre, apellido, fecha de nacimiento, número de carnet de identidad
- Correo electrónico, número de teléfono
- Fotografía de perfil y fotografía de verificación de identidad
- Datos del vehículo (conductores): placa, marca, modelo, año, tipo de combustible

### Datos recopilados automáticamente:
- **Ubicación en tiempo real** (GPS): necesaria para el funcionamiento del servicio. Solo se recopila cuando la app está activa y el usuario ha autorizado el permiso.
- Historial de viajes y rutas recorridas
- Datos de uso de la aplicación
- Dirección IP y datos del dispositivo

### Datos biométricos:
- Fotografía facial para verificación de identidad (reconocimiento facial)
- Esta información se procesa localmente y no se comparte con terceros

## 2. Cómo Usamos los Datos

- Verificar la identidad de los usuarios (seguridad)
- Facilitar la conexión entre conductores y pasajeros
- Calcular rutas óptimas y precios de viaje
- Procesar pagos y gestionar el sistema de coins
- Enviar notificaciones relacionadas con el servicio
- Mejorar el servicio mediante análisis de uso (machine learning interno)
- Cumplir obligaciones legales

## 3. Compartición de Datos

Apuradito NO vende datos personales a terceros. Solo compartimos información con:
- Otros usuarios de la plataforma: el conductor ve el nombre y foto del pasajero y viceversa (solo datos necesarios para el viaje)
- Autoridades bolivianas: cuando sea requerido por ley
- Proveedores tecnológicos: Firebase (Google) para notificaciones push y autenticación (sujeto a sus propias políticas de privacidad)

## 4. Retención de Datos

- Datos de cuenta activa: mientras la cuenta esté activa
- Datos de cuenta eliminada: 90 días después de la eliminación
- Historial de viajes: 5 años (por obligaciones fiscales)
- Datos biométricos: hasta 30 días después de completar la verificación

## 5. Derechos del Usuario

El usuario tiene derecho a:
- Acceder a sus datos personales
- Rectificar datos incorrectos
- Solicitar la eliminación de su cuenta y datos
- Revocar el consentimiento para el uso de ubicación
- Exportar sus datos en formato legible

Para ejercer estos derechos, escribir a: privacidad@apuradito.bo

## 6. Seguridad

- Los datos se transmiten cifrados (HTTPS/TLS)
- Las contraseñas se almacenan con hash bcrypt (no se guardan en texto plano)
- Los tokens de sesión tienen expiración automática
- El acceso a datos de producción está restringido al equipo técnico

---
*Acepto que Apuradito recopile y procese mis datos personales conforme a esta Política de Privacidad.*
