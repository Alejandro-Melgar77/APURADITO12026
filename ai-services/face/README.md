# Apuradito Facial Service

API FastAPI para detección y verificación facial. Publica `POST /v1/facial/detect`, `POST /v1/facial/verify` y `GET /health`.

La verificación biométrica debe usarse con consentimiento explícito y revisión humana para resultados ambiguos.
