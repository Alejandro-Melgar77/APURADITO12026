COMANDOS A UTILIZAR 

DOCKER:
Asegúrate de tener Docker Desktop abierto y ejecutándose en segundo plano. Luego, abre una terminal en la raíz de tu proyecto (APURADITO) y ejecuta:

docker-compose up -d

BACKEND:

cd backend
copy .env.example .env
pip install -r requirements.txt
uvicorn app.main:app --reload --port 8000


Abre otra terminal en la carpeta backend e inserta los datos semilla de Santa Cruz de la Sierra (conductores, vehículos y variables de precio):
python -m seeds.run_all


FRONTEND:

cd frontend
npm install
npm run dev
